# Capstone-Project--AI_ML
Applied AI &amp; ML Capstone Project — End-to-End Data Intelligence System
## 📊 Dataset Declaration & Justification

### Dataset Declaration
* **Name:** Students Habits and Performance Dataset
* **Source:** Kaggle datasets in CSV format
* **Format:** CSV (Comma-Separated Values)
* **Key Features:** [study_hours_per_day, social_media_hours, netflix_hours, attendance_percentage, part_time_job, etc.]

### Justification: Why This Dataset?
Understanding the direct relationship between a student's daily routines and their academic success is critical for his/her own life.

I selected this dataset for two primary reasons:
1. **Behavioral Insight:** It captures both lifestyle choices (habits like sleep, screen time, or study intervals) and concrete outcomes (grades/performance). 
2. **Predictive Modeling Potential:** The data contains a healthy mix of continuous variables (like study_hours_per_day, social_media_hours) and categorical factors (gender, part_time_job).
